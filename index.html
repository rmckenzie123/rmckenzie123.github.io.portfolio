Rock Paper Scissors

CSS Focussed

This is an interactive game that takes the users input and replies with predetermined responses and keep track of the score.

The game itself has been created using javascript, by defining which item (rock, paper and scissors) is greater than the other (eg paper beats rock), the computer randomly generates one of the options and the user selects theres, this is then compered using a switch statement which contains each variation of either a win lose or draw, this is the relayed back the user and the score is updated.

As a front end project the design of this was the first priority, as it’s a simple childish game I made the design match this with the font looking handwritten, light colours and a school themed background.

Some interesting features of the CSS is the slow fade of the buttons which uses the choice:hover feature and ‘transition: all 0.5s ease;’ which makes the colour fade nice and smooth. The HTML is very simple and only has a few lines of code

Snake

JavaScript Focussed

As my first project ‘Rock Paper Scissors’ was quite simple in terms of the Javascript and the game itself, I wanted to make my second project more complex and engaging.

The area in which the game is played uses the ‘canvas’ property of javascript, which I use as a grid, the direction is defined by the x and y axis and the snake grows a block and deletes a block which creates the appearance that the snake is moving.

The apple is randomly generated and when the snakes position of the grid is equal to the position of the food the snake skips a block deletion making the snake grow and generates another apple.

If the position of the snake is equal to the border or the snakes tail then the length of the snakes tail is set back to 0.
